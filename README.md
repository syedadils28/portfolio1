# Mohammad Ruhul Ameen — Portfolio

## File Structure
```
portfolio/
├── index.html          ← Main HTML file
├── css/
│   └── style.css       ← All styles
├── js/
│   └── main.js         ← Typing effect, scroll animations, form logic
├── assets/
│   ├── resume.pdf      ← Place your resume here
│   ├── cert1.jpg       ← Certificate images
│   ├── cert2.jpg
│   └── cert3.jpg
└── README.md
```

## How to add your certificate images
Replace the `.cert-placeholder` div in index.html with:
```html
<img src="assets/cert1.jpg" class="cert-image" alt="Certificate Name" />
```

## How to use
Open `index.html` in any browser — no build step needed.
